# 🏛️ CoreBrain Knowledge Base

Este diretório contém a inteligência destilada e purificada do ecossistema FourCoders. É o nosso "Banco de Dados Vetorial" manual que alimenta todos os projetos periféricos.

## 📂 Estrutura de Conhecimento
- `/architecture`: Padrões de design de sistemas, diagramas C4 e decisões estruturais.
- `/patterns`: Snippets de código de elite, componentes UI "Wow" e lógicas de backend reutilizáveis.
- `/prompts`: Versões otimizadas de instruções (Neuro-Optimizer) para diferentes tarefas.
- `/lessons`: Erros superados e aprendizados técnicos de projetos reais.

## 🧬 Como alimentar este banco
O conhecimento é colhido através da skill **Geneticist** ao final de cada tarefa relevante em qualquer projeto.

## 🔍 Como usar
Sempre que iniciar um novo projeto, peça ao Brain para:
1. "Escaneie a pasta /knowledge para entender nossos padrões."
2. "Aplique a inteligência de /patterns/ [nome] nesta funcionalidade."
